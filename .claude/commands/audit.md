# /audit — Design Quality Check

Run a comprehensive audit against all installed skills:

## Checklist:
- [ ] Font is NOT Inter/Roboto/Open Sans/Arial
- [ ] No pure black on pure white
- [ ] No purple/blue gradient backgrounds
- [ ] No bounce/elastic easing on UI
- [ ] Spacing follows a modular scale
- [ ] Colors use tinted neutrals
- [ ] Custom easing on all transitions
- [ ] Scroll animations present and purposeful
- [ ] Hover/focus states on all interactive elements
- [ ] Accessible (WCAG 2.1 AA minimum)
- [ ] Performance: animations use transform/opacity only
- [ ] No cards-inside-cards nesting
- [ ] Asymmetric layout with clear hierarchy
- [ ] Mobile responsive with distinct mobile experience

Score each item 0-10 and provide an overall score.
Anything below 7 gets specific fix recommendations.
