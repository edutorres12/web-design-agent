# /design — Create a New Design

Follow the complete design pipeline from CLAUDE.md.

## Steps:
1. Ask what type of site/page to design (if not specified)
2. **Direction Phase**: Read impeccable, ui-ux-pro-max, color-expert, google-fonts skills
3. Present 2-3 aesthetic directions with:
   - Font pairing
   - Color palette (5 colors: primary, accent, 2 neutrals, background)
   - Layout approach
   - Motion strategy
4. Wait for user approval on direction
5. **Build Phase**: Create the HTML/CSS/JS
6. **Polish Phase**: Run audit, fix issues
7. Present the final result

Remember: NEVER output generic-looking code. Every design decision must be intentional.
