# /polish — Refine Existing Design

Read the current code and apply refinements:

1. **Typography audit** (skills/impeccable, skills/google-fonts)
   - Is the font pairing distinctive?
   - Are sizes following a modular scale?
   - Is line-height optimized for readability?

2. **Color audit** (skills/color-expert)
   - Are neutrals tinted (not pure gray)?
   - Is contrast sufficient (APCA)?
   - Is the palette cohesive?

3. **Spacing audit** (skills/interface-design)
   - Following a consistent scale?
   - Purposeful whitespace?

4. **Motion audit** (skills/ui-motion-perf, skills/gsap-skills)
   - Are transitions smooth (60fps)?
   - Custom easing curves?
   - Scroll animations enhance story?

5. **Micro-interactions** (skills/design-with-claude)
   - Hover states on interactive elements?
   - Focus states for accessibility?
   - Loading/transition states?

Apply fixes and explain each change.
