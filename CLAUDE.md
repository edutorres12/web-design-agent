## Design Pipeline

### Phase 1: Direction (BEFORE writing any code)

**Goal: Establish a distinctive visual direction that could never be mistaken for AI-generated.**

1. **Anti-generic foundation:**
   Read `skills/impeccable/.claude/skills/frontend-design/SKILL.md`
   Read `skills/anthropic-skills/skills/frontend-design/SKILL.md`
   → Internalize what makes designs look generic vs. distinctive.

2. **Choose a UI style:**
   Read `skills/ui-ux-pro-max/.claude/skills/ui-styling/SKILL.md`
   → Pick from 84 styles (brutalist, editorial, art-deco, etc.) — NOT "clean modern".

3. **Color palette:**
   Read `skills/color-expert/SKILL.md`
   Read `skills/impeccable/.claude/skills/colorize/SKILL.md`
   → Use OKLCH. Tinted neutrals. NEVER pure gray. NEVER default blue.

4. **Typography:**
   Read `skills/google-fonts/SKILL.md`
   Read `skills/impeccable/.claude/skills/typeset/SKILL.md`
   Read `skills/design-for-ai/commands/fonts.md`
   → Pick a distinctive pairing. BANNED: Inter, Roboto, Open Sans, Arial, Helvetica.

5. **Design tokens:**
   Read `skills/interface-design/.claude/skills/interface-design/SKILL.md`
   Read `skills/impeccable/.claude/skills/extract/SKILL.md`
   → Establish spacing scale, color tokens, type scale BEFORE building.

6. **Brand identity (if applicable):**
   Read `skills/anthropic-skills/skills/brand-guidelines/SKILL.md`
   → Apply brand colors, voice, visual language.

7. **Validate direction:**
   Read `skills/design-for-ai/skills/design-for-ai/SKILL.md`
   → Run CHECKER mode against your choices. Fix before proceeding.

**OUTPUT: Present 2-3 directions to the user with font, colors, layout, and motion strategy.**

## Anti-Patterns (NEVER do these)

### Typography
- Inter, Roboto, Open Sans, Arial, Helvetica as primary font
- Same font weight everywhere (use contrast: 300 vs 700)
- Default line-height (use 1.2 for headings, 1.5-1.6 for body)
- Font sizes that don't follow a scale

### Color
- Pure black (#000) on pure white (#fff)
- Purple/blue gradient backgrounds
- Gray neutrals (use tinted: warm gray, cool slate)
- More than 2 accent colors
- Default link blue (#0000FF)

### Layout
- Everything centered with no hierarchy
- Cards inside cards inside cards
- Equal spacing everywhere
- Same border-radius on everything (8px syndrome)
- Symmetric grids with no visual tension

### Motion
- Bounce easing or elastic easing on UI elements
- Animations longer than 600ms for UI interactions
- Parallax on text (ruins readability)
- Motion without purpose (decorative-only)
- No motion at all (feels static/dead)

### Content
- "Lorem ipsum" placeholder text
- Generic stock photo heroes
- Emojis in professional UI
- "Click here" or "Learn more" as link text

## What Premium Looks Like

- **Typography:** Distinctive font pairing (display + body), modular type scale, optical kerning
- **Color:** Tinted neutrals, OKLCH color space, 5-shade palette (primary, accent, 2 neutrals, bg)
- **Space:** Consistent scale (4, 8, 12, 16, 24, 32, 48, 64, 96), purposeful whitespace
- **Motion:** Custom easing `cubic-bezier(0.25, 0.1, 0.25, 1)`, scroll reveals, staggered entrances
- **Layout:** Asymmetric with clear flow, visual tension, intentional density changes
- **Depth:** Subtle layered shadows, not flat and not skeuomorphic
- **Details:** Custom cursors, selection colors, scrollbar styling, focus states, hover transitions
